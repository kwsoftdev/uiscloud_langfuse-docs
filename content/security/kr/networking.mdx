Langfuse Cloud는 기본적으로 공개 리전 엔드포인트를 사용합니다. AWS PrivateLink는 계약을 체결한 Enterprise 고객에게 제공되는 선택적 기능이며, 리전별 가용성과 온보딩 요구 사항에 따라 달라질 수 있습니다. 해당 리전에서의 사용 가능 여부를 논의하려면 [영업팀에 문의](/talk-to-us)해 주세요.

프라이빗 연결과 조직에서 완전히 제어하는 인프라가 필요하다면 [Langfuse 셀프 호스팅](/self-hosting)을 고려해 보세요. 보안 요구 사항에 가장 적합한 솔루션을 논의하려면 [저희와 상담 일정을 예약](/talk-to-us)하세요.

## Langfuse Cloud 아웃바운드(Egress) IP 주소

Langfuse Cloud는 LLM-as-a-judge 평가나 블롭 스토리지 내보내기와 같은 기능을 위해 사용자를 대신하여 저희 인프라에서 아웃바운드 API 호출을 수행합니다. 보안을 강화하려면 방화벽이나 보안 그룹 설정에서 이 특정 연결을 허용하도록 아래 IP 주소를 화이트리스트에 등록하는 것이 좋습니다.
