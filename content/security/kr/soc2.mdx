# SOC 2 Type II 준수

Langfuse Cloud는 **SOC 2 Type II 감사**를 성공적으로 완료했습니다.

- **상태:** 인증 완료
- **보고서 제공:** SOC 2 Type II 보고서는 Pro, Team 또는 Enterprise 플랜을 사용하는 고객에게 요청 시 제공됩니다.
- **접근 요청:** [여기](/request-security-docs)에서 보고서 접근을 요청할 수 있습니다.
