# 프롬프트 컴포저빌리티

더 많은 프롬프트를 만들다 보면, 여러 프롬프트에서 동일한 텍스트 조각이나 지침을 반복해서 사용하게 되는 경우가 많습니다. 중복을 피하기 위해 다른 프롬프트를 참조하여 프롬프트를 구성(compose)할 수 있습니다.

<Video
  src="/images/docs/prompt-composition.mp4"
  aspectRatio={1030 / 712}
  gifStyle
  className="rounded-lg max-w-lg"

/>

## 컴포즈드 프롬프트를 사용하는 이유

- 여러 프롬프트에서 재사용할 수 있는 모듈식 **프롬프트 컴포넌트**를 만들 수 있습니다
- 공통 지침, 예시, 컨텍스트를 한곳에서 **유지 관리**할 수 있습니다
- 기본 프롬프트가 변경되면 이를 참조하는 프롬프트가 자동으로 **업데이트**됩니다

## 시작하기

<LangTabs items={["Langfuse UI", "SDKs/API"]}>

<Tab>
Langfuse UI를 통해 프롬프트를 생성할 때 `Add prompt reference` 버튼을 사용하여 프롬프트에 프롬프트 참조를 삽입할 수 있습니다.

<Video
  src="https://static.langfuse.com/docs-videos/prompt-composability.mp4"
  aspectRatio={16 / 9}
  gifStyle
/>

</Tab>

<Tab>

다음 형식을 사용하여 프롬프트에서 다른 **텍스트** 프롬프트를 참조할 수 있습니다.

```bash
@@@langfusePrompt:name=PromptName|version=1@@@
```

특정 버전 대신 레이블을 사용하여 동적으로 참조를 해석하도록 할 수도 있습니다.

```bash
@@@langfusePrompt:name=PromptName|label=production@@@
```

</Tab>

</LangTabs>

원하는 기능이 아닌가요? 다음과 같은 유사한 기능을 참고하세요.

- 프롬프트에 동적 텍스트를 삽입하려면 [변수](/docs/prompt-management/features/variables)
- 문자열 대신 완전한 메시지 배열을 삽입하려면 [메시지 플레이스홀더](/docs/prompt-management/features/message-placeholders)
