# 프롬프트 폴더

프롬프트 폴더를 사용하면 프롬프트를 논리적인 그룹으로 정리할 수 있습니다. 프롬프트 라이브러리가 커짐에 따라, 폴더를 사용하면 기능, 팀, 환경 또는 워크플로우에 적합한 구조에 따라 관련 프롬프트를 함께 유지할 수 있습니다.

폴더를 만들려면 프롬프트 이름에 슬래시(`/`)를 추가하세요. UI는 `/`로 끝나는 모든 세그먼트를 자동으로 폴더로 표시합니다. 이런 방식으로 폴더 구조를 구성할 수 있습니다.

<Callout type="info">

**참고**: Python SDK를 통해 폴더 내 프롬프트에 접근하려면 `langfuse >= 3.0.2`가 필요합니다.

</Callout>
