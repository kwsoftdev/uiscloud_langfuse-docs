# Langfuse 프롬프트 관리를 위한 n8n 노드

Langfuse n8n 노드는 [Langfuse의 프롬프트 관리](/docs/prompts/get-started)를 n8n 워크플로우에 원활하게 통합할 수 있게 해줍니다. 이 커뮤니티 관리형 노드를 사용하면 n8n 워크플로우 내에서 Langfuse 프로젝트의 프롬프트를 직접 가져와 사용할 수 있습니다.

> **n8n이란 무엇인가요?** [n8n](https://github.com/n8n-io/n8n)은 전체 코드를 작성하지 않고도 API, 앱, 데이터를 시각적으로 연결하고 오케스트레이션할 수 있는 오픈소스 노드 기반 워크플로우 자동화 플랫폼입니다.

## 설치

셀프 호스팅 n8n: 패키지 이름 [`@langfuse/n8n-nodes-langfuse`](https://www.npmjs.com/package/@langfuse/n8n-nodes-langfuse)를 사용하여 **Settings** > **Community Nodes**를 통해 설치합니다.

n8n Cloud: 워크플로우에서 `Langfuse`를 검색하여 노드를 바로 사용할 수 있습니다.
