# Langfuse 개요

Langfuse는 팀이 LLM 애플리케이션을 공동으로 디버깅, 분석, 개선할 수 있도록 돕는 오픈소스 AI 엔지니어링 플랫폼([GitHub](https://github.com/langfuse/langfuse))입니다. 모든 플랫폼 기능은 개발 워크플로를 가속화하기 위해 네이티브로 통합되어 있습니다. Langfuse는 개방적이고, 셀프 호스팅이 가능하며, 확장 가능합니다([_langfuse를 선택하는 이유_](/why)).

## 관측성 [#observability]

[관측성](/docs/observability/overview)은 LLM 애플리케이션을 이해하고 디버깅하는 데 필수적입니다. 전통적인 소프트웨어와 달리 LLM 애플리케이션은 모니터링하고 디버깅하기 어려운 복잡하고 비결정적인 상호작용을 포함합니다. Langfuse는 애플리케이션에서 정확히 무슨 일이 일어나고 있는지 이해할 수 있도록 포괄적인 트레이싱 기능을 제공합니다.

- 트레이스에는 검색, 임베딩, API 호출 등을 포함한 모든 LLM 및 비LLM 호출이 포함됩니다
- 세션으로서의 멀티턴 대화 추적 및 사용자 추적을 지원합니다
- 에이전트를 그래프로 표현할 수 있습니다
- Python/JS용 네이티브 SDK, 100개 이상의 라이브러리/프레임워크 통합, OpenTelemetry, 또는 LiteLLM과 같은 LLM 게이트웨이를 통해 트레이스를 캡처할 수 있습니다
- 호환성을 높이고 벤더 종속을 줄이기 위해 OpenTelemetry를 기반으로 합니다

예시를 확인하고 싶으신가요? [인터랙티브 데모](/docs/demo)를 사용해 보세요.

## 프롬프트 관리 [#prompts]

[프롬프트 관리](/docs/prompt-management/overview)는 효과적인 LLM 애플리케이션을 구축하는 데 매우 중요합니다. Langfuse는 개발 라이프사이클 전반에 걸쳐 프롬프트를 관리, 버전 관리, 최적화할 수 있는 도구를 제공합니다.

- 프롬프트 관리 [시작하기](/docs/prompt-management/get-started)
- 개발 라이프사이클 전반에 걸쳐 프롬프트를 관리, 버전 관리, 최적화합니다
- [LLM 플레이그라운드](/docs/prompt-management/features/playground)에서 프롬프트를 대화형으로 테스트합니다
- Langfuse 내에서 새 프롬프트 버전을 직접 테스트하기 위해 데이터셋에 대해 [실험](/docs/evaluation/features/prompt-experiments)을 실행합니다

## 평가 [#evaluation]

[평가](/docs/evaluation/overview)는 LLM 애플리케이션의 품질과 신뢰성을 보장하는 데 매우 중요합니다. Langfuse는 개발 단계에서 테스트하든 프로덕션 성능을 모니터링하든, 특정 요구 사항에 맞게 조정되는 유연한 평가 도구를 제공합니다.

- LLM-as-a-judge, 코드 평가기, 사용자 피드백, 수동 라벨링, 또는 커스텀 파이프라인 등 다양한 [평가 방법](/docs/evaluation/overview)으로 시작합니다
- 프로덕션 트레이스에 대해 평가를 실행하여 문제를 조기에 파악합니다
- 애플리케이션이 다양한 시나리오에서 안정적으로 작동하도록 보장하는 개발 단계의 체계적인 테스트를 위해 [데이터셋](/docs/evaluation/features/datasets)을 생성하고 관리합니다
- LLM 애플리케이션을 체계적으로 테스트하기 위해 [실험](/docs/evaluation/core-concepts#experiments)을 실행합니다

더 알아보고 싶으신가요? Langfuse 평가에 대한 [**엔드투엔드 워크스루 보기**](/watch-demo?tab=evaluation)와 이를 활용해 LLM 애플리케이션을 개선하는 방법을 확인해 보세요.

## Langfuse를 선택하는 이유

- **오픈소스:** 커스텀 통합을 위한 공개 API를 갖춘 완전한 오픈소스
- **프로덕션 최적화:** 최소한의 성능 오버헤드를 고려하여 설계
- **최고 수준의 SDK:** Python 및 JavaScript용 네이티브 SDK
- **프레임워크 지원:** OpenAI SDK, LangChain, LlamaIndex 등 인기 있는 프레임워크와 통합
- **멀티모달:** 텍스트, 이미지 및 기타 모달리티의 트레이싱 지원
- **완전한 플랫폼:** LLM 애플리케이션 개발 라이프사이클 전반을 위한 도구 모음
