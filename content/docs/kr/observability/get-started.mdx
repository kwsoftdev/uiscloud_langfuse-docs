# 트레이싱 시작하기

이 가이드는 Langfuse에 첫 번째 트레이스를 수집하는 과정을 안내합니다. 트레이싱이 무엇이고 왜 중요한지 먼저 이해하고 싶다면 [관측성 개요](/docs/observability/overview)를 먼저 확인하세요. Langfuse에서 트레이스가 어떻게 구성되고 내부적으로 어떻게 동작하는지에 대한 자세한 내용은 [핵심 개념](/docs/observability/data-model)을 참고하세요.

## 에이전틱 설치 [#agentic-installation]

[Langfuse 에이전트 스킬](https://github.com/langfuse/skills)을 설치하면 코딩 에이전트가 모든 Langfuse 기능에 접근할 수 있습니다.

<Tabs items={["Ask your coding agent", "Cursor plugin", "Manual installation"]}>

<Tab>

코딩 에이전트에게 [GitHub 저장소](https://github.com/langfuse/skills)를 알려주고 이를 이용해 트레이싱을 시작하도록 요청하세요.

```txt filename="Agent instruction"
Install the Langfuse Agent Skill from github.com/langfuse/skills
and use it to add tracing to this application with Langfuse
following best practices.
```

</Tab>

<Tab>

Langfuse에는 이 스킬을 자동으로 포함하는 [Cursor 플러그인](https://cursor.com/docs/plugins)이 있습니다.

<div className="flex gap-2 mt-3 mb-6">
  <Button asChild>
    <Link
      href="https://cursor.com/marketplace/langfuse"
      target="_blank"
      rel="noopener noreferrer"
    >
      Install Plugin in Cursor
    </Link>
  </Button>
</div>

그런 다음 에이전트에게 다음과 같이 요청하세요.

```txt filename="Agent instruction"
Add tracing to this application with Langfuse following best practices.
```

</Tab>

<Tab>

npm을 통해 설치합니다 ([skills CLI](https://www.npmjs.com/package/skills)).

```bash
npx skills add langfuse/skills --skill "langfuse"
```

특정 에이전트를 직접 대상으로 지정하려면 다음과 같이 하세요.

```bash
npx skills add langfuse/skills --skill "langfuse" --agent "<agent-id>"
```

<Details>
<Summary>Alternatively you can manually clone the skill</Summary>

1. 안정적인 위치에 저장소를 클론합니다

```bash
git clone https://github.com/langfuse/skills.git /path/to/langfuse-skills
```

2. 에이전트의 스킬 디렉터리가 존재하는지 확인합니다

```bash
mkdir -p /path/to/<agent-skill-root>/skills
```

3. 스킬 폴더를 심볼릭 링크로 연결합니다

```bash
ln -s /path/to/langfuse-skills/skills/langfuse /path/to/<agent-skill-root>/skills/langfuse
```

</Details>

그런 다음 에이전트에게 다음과 같이 요청하세요.

```txt filename="Agent instruction"
Add tracing to this application with Langfuse following best practices.
```

</Tab>

</Tabs>

## 수동 설치 [#manual-installation]

이 가이드는 Langfuse 트레이싱을 수동으로 시작하는 데 도움이 됩니다.

<Steps>
### API 키 발급받기

1.  [Langfuse 계정을 생성](https://cloud.langfuse.com/auth/sign-up)하거나 [Langfuse를 셀프 호스팅](/self-hosting)하세요.
2.  프로젝트 설정에서 새 API 자격 증명을 생성하세요.

### 첫 번째 트레이스 수집하기

프레임워크, SDK, 또는 OpenTelemetry 설정 중 원하는 것을 선택해 시작하세요. 애플리케이션이 이미 OTEL 스팬을 내보내고 있다면 OpenTelemetry 가이드부터 시작하세요.
