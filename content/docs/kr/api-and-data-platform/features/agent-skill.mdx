# Langfuse 에이전트 스킬

Langfuse 에이전트 스킬은 AI 코딩 에이전트가 Langfuse를 효과적으로 사용할 수 있도록 도와줍니다. 이 스킬은 공개된 [Agent Skills](https://github.com/anthropics/skills) 표준을 따르며, Claude Code, Cursor, Windsurf 및 기타 호환 가능한 에이전트와 함께 작동합니다. 이 스킬은 [GitHub](https://github.com/langfuse/skills)에서 오픈소스로 제공됩니다.

## 사용하는 이유 [#why-use-it]

코딩 에이전트는 이 스킬이 설치되어 있으면 모범 사례를 따르도록 조건화되기 때문에 훨씬 더 나은 결과를 만들어냅니다. 스킬에 접근할 수 있는 코딩 에이전트는 Langfuse의 지식을 기반으로 좋은 결과물이 어떤 모습이어야 하는지에 대한 확고한 관점을 갖게 됩니다.

이 스킬은 일반적인 규칙과 문서 사용 지침을 설명하는 `SKILL.md` 진입점과, 특정 워크플로우를 위한 구체적인 모범 사례로 채워진 참조 문서로 구성된 독립적인 폴더입니다.

이 스킬은 점진적 공개(progressive disclosure) 모델을 사용합니다. 프론트매터는 항상 에이전트의 컨텍스트에 로드되어 언제 이 스킬이 관련이 있는지 알 수 있게 하지만, 전체 지침과 참조 문서는 필요할 때만 로드됩니다. 이를 통해 컨텍스트 사용량을 낮게 유지하면서도 에이전트가 전문 지식에 접근할 수 있게 합니다.

- _src/agent.ts의 시스템 프롬프트를 Langfuse 프롬프트 관리로 마이그레이션해줘_
