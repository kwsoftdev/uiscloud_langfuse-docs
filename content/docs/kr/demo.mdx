# 예제 프로젝트

import { Button } from "@/components/ui/button";
import { ToAppButton } from "@/components/ToAppButton";

Langfuse 예제 프로젝트는 자신의 계정을 설정하기 전에 실제 데이터를 통해 Langfuse의 기능을 살펴볼 수 있는 **실시간 공유 프로젝트**입니다. 여러 팀이 LLM 관측성, 프롬프트 관리, 평가를 위해 Langfuse를 어떻게 사용하는지 직접 체험할 수 있는 실습형 둘러보기라고 생각하시면 됩니다.

예제 프로젝트는 **보기 전용 액세스**를 제공합니다.

## 예제 프로젝트 시작하기

### 1단계: 예제 프로젝트 액세스하기

예제 프로젝트에 액세스하려면 무료 계정을 만드세요(신용카드가 필요하지 않습니다).

<div className="mt-3 not-prose flex">
  <ToAppButton
    signedInText="View Example project"
    signUpText="Create Example account"
    dropdownText="View Example project"
  />
</div>

### 2단계: 화면 이해하기

예제 프로젝트를 처음 열면 **Traces** 페이지가 표시됩니다. 이 페이지에서 볼 수 있는 내용은 다음과 같습니다.

- 각 행은 예제 챗봇과의 하나의 상호작용을 나타냅니다
- 자신의 트레이스뿐만 아니라 모든 사용자의 트레이스가 표시됩니다. 다양한 예시를 살펴볼 수 있도록 의도적으로 이렇게 구성되어 있습니다
- 트레이스에는 타이밍, 비용, 입력/출력, 그리고 평가에 의해 부여된 점수가 표시됩니다

**직접 해보세요:**

1. 트레이스를 클릭하여 상세한 실행 단계를 확인해보세요
2. 챗봇의 구성 요소들이 어떻게 함께 작동하는지 보여주는 그래프 뷰를 살펴보세요
3. 점수가 있는 트레이스를 찾아 평가가 어떻게 작동하는지 확인해보세요

**모든 기능 살펴보기:** 왼쪽 내비게이션을 통해 [트레이싱](/docs/observability/overview), [세션](/docs/observability/features/sessions), [프롬프트](/docs/prompt-management/overview), [점수](/docs/evaluation/overview), [데이터셋](/docs/evaluation/experiments/datasets)을 살펴보세요. 각 영역은 완전한 LLM 애플리케이션에서 Langfuse가 어떻게 작동하는지 보여줍니다.
