# 웨비나: 멀티 스텝 LLM 시스템의 추적 가능성과 관측성

이번 세션에서는 멀티 스텝 LLM 시스템에서 추적 가능성(traceability)과 관측성(observability)을 구현하기 위한 고급 기법과 모범 사례에 대해 이야기했습니다.

다룬 주제:

- 멀티 스텝 LLM 애플리케이션: 여러 단계, 에이전트, 체인을 포함하는 LLM 시스템의 복잡성을 이해합니다.
- 통합 예시 (RAG): RAG 시스템이 어떻게 동작하는지 자세히 살펴봅니다. 사용자 입력, 메시지 히스토리, 추가 메모리가 어떻게 결합되어 출처가 함께 제공되는 통합 응답을 생성하는지 시연합니다.
- 관측성을 위한 트레이스: 관측성에서 트레이스가 하는 역할과, 프로덕션 및 개발 환경 모두에서 로깅 및 모니터링을 위한 강력한 추상화로서 어떻게 활용되는지 알아봅니다.
