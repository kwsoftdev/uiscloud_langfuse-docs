# 데이터셋 실험을 위한 LLM-as-a-Judge 평가기

Langfuse의 새로운 매니지드 LLM-as-a-judge 평가기를 사용하여 LLM 애플리케이션 변경 사항을 안정적으로 평가하는 방법을 소개하는 10분 분량의 안내 영상입니다.

이 기능은 팀이 다음과 같은 작업을 수행하는 데 도움을 줍니다:

- 테스트 데이터셋을 기준으로 실험 실행을 자동으로 평가
- 서로 다른 버전 간의 메트릭 비교
- 프로덕션에 반영되기 전에 회귀(regression) 식별
- 환각(hallucination), 유용성, 관련성 등의 기준에 따라 출력에 점수 부여

함수 호출(function calling)을 통해 OpenAI, Anthropic, Azure OpenAI, AWS Bedrock 등 널리 사용되는 LLM 제공업체와 함께 작동합니다.
