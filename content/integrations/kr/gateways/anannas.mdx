# Anannas AI 통합

이 가이드에서는 [Langfuse](/)를 [Anannas AI](https://anannas.ai)와 통합하는 방법을 보여드립니다.

> **Anannas AI란?** [Anannas AI](https://anannas.ai)는 OpenAI 호환 API를 통해 500개 이상의 모델(OpenAI, Anthropic, Mistral, Gemini, DeepSeek 등)에 접근할 수 있는 통합 추론 게이트웨이입니다. 캐시 적중률 분석, 토큰 단위 메트릭, 도구 호출 분석, 모델 효율성 점수 등 내장 관측 기능을 제공하며, 자동 폴백 라우팅을 지원하는 프로바이더 상태 모니터링, 약 0.48ms의 오버헤드와 5%의 마크업, 그리고 엔터프라이즈 배포를 위한 BYOK(Bring Your Own Key) 지원을 제공합니다.

> **Langfuse란?** [Langfuse](/)는 팀이 LLM 호출을 트레이싱하고, 성능을 모니터링하며, AI 애플리케이션의 문제를 디버그하도록 돕는 오픈소스 AI 엔지니어링 플랫폼입니다.

Anannas AI는 OpenAI API 스키마를 사용하므로, [Python](/integrations/model-providers/openai-py)과 [TypeScript](/integrations/model-providers/openai-js) 모두에서 제공되는 Langfuse의 OpenAI SDK 네이티브 통합을 활용할 수 있습니다.

## 시작하기

```bash
pip install langfuse openai
```

```python
import os

# Set your Langfuse API keys
LANGFUSE_SECRET_KEY="sk-lf-..."
LANGFUSE_PUBLIC_KEY="pk-lf-..."
LANGFUSE_BASE_URL="https://cloud.langfuse.com" # 🇪🇺 EU region
# Other Langfuse data regions include 🇺🇸 US: https://us.cloud.langfuse.com, 🇯🇵 Japan: https://jp.cloud.langfuse.com and ⚕️ HIPAA: https://hipaa.cloud.langfuse.com
# Set your Anannas API key
os.environ["ANANNAS_API_KEY"] = "<YOUR_ANANNAS_API_KEY>"
```

## 예제 1: 단순 LLM 호출

Anannas AI는 OpenAI 호환 API를 제공하므로, [Langfuse OpenAI SDK 래퍼](/integrations/model-providers/openai-py)를 사용해 Anannas AI 호출을 Langfuse의 생성(generation)으로 자동 기록할 수 있습니다.

- `base_url`은 Anannas AI의 API 엔드포인트로 설정됩니다.
- `"anthropic/claude-3-5-sonnet"`을 Anannas AI에서 제공하는 500개 이상의 모델 중 원하는 모델로 바꿀 수 있습니다.
- API 키는 `ANANNAS_API_KEY` 환경 변수에서 읽어옵니다.

```python
# Import the Langfuse OpenAI SDK wrapper
from langfuse.openai import openai

# Create an OpenAI client with Anannas AI's base URL
client = openai.OpenAI(
    api_key=os.environ["ANANNAS_API_KEY"],
    base_url="https://api.anannas.ai/v1"
)

# Make a chat completion request
response = client.chat.completions.create(
    model="anthropic/claude-3-5-sonnet",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Tell me a fun fact about pineapples."}
    ],
    name="fun-fact-request"  # Optional: Name of the generation in Langfuse
)

# Print the assistant's reply
print(response.choices[0].message.content)
```

## 예제 2: 중첩된 LLM 호출

`@observe()` 데코레이터를 사용하면, 중첩된 LLM 호출, 입력, 출력, 실행 시간을 포함해 모든 Python 함수의 실행 세부 정보를 캡처할 수 있습니다. 이를 통해 최소한의 코드 변경으로 심층적인 관측성을 확보할 수 있습니다.

- `@observe()` 데코레이터는 함수의 입력, 출력, 실행 세부 정보를 캡처합니다.
- 중첩된 함수 `summarize_text`와 `analyze_sentiment`에도 데코레이터가 적용되어 트레이스 계층 구조가 만들어집니다.
- 함수 내의 각 LLM 호출이 기록되어, 실행 흐름의 상세한 트레이스를 제공합니다.

```python
from langfuse import observe
from langfuse.openai import openai

# Create an OpenAI client with Anannas AI's base URL
client = openai.OpenAI(
    api_key=os.environ["ANANNAS_API_KEY"],
    base_url="https://api.anannas.ai/v1"
)

@observe()  # This decorator enables tracing of the function
def analyze_text(text: str):
    # First LLM call: Summarize the text
    summary_response = summarize_text(text)
    summary = summary_response.choices[0].message.content

    # Second LLM call: Analyze the sentiment of the summary
    sentiment_response = analyze_sentiment(summary)
    sentiment = sentiment_response.choices[0].message.content

    return {
        "summary": summary,
        "sentiment": sentiment
    }

@observe()  # Nested function to be traced
def summarize_text(text: str):
    return client.chat.completions.create(
        model="openai/gpt-4o",
        messages=[
            {"role": "system", "content": "You summarize texts in a concise manner."},
            {"role": "user", "content": f"Summarize the following text:\n{text}"}
        ],
        name="summarize-text"
    )

@observe()  # Nested function to be traced
def analyze_sentiment(summary: str):
    return client.chat.completions.create(
        model="openai/gpt-4o-mini",
        messages=[
            {"role": "system", "content": "You analyze the sentiment of texts."},
            {"role": "user", "content": f"Analyze the sentiment of the following summary:\n{summary}"}
        ],
        name="analyze-sentiment"
    )

# Example usage
text_to_analyze = "Anannas AI provides a unified gateway to access hundreds of LLM models with built-in observability and automatic fallback routing."
analyze_text(text_to_analyze)
```
