# Portkey LLM 게이트웨이를 위한 Langfuse 관측성

이 가이드에서는 Portkey의 AI 게이트웨이를 Langfuse와 통합하는 방법을 보여드립니다. Portkey의 API 엔드포인트는 OpenAI SDK와 완전히 [호환](https://portkey.ai/docs/api-reference/inference-api/introduction)되므로, AI 애플리케이션을 원활하게 트레이싱하고 모니터링할 수 있습니다.

> **Portkey란?** [Portkey](https://portkey.ai/)는 250개 이상의 AI 모델과 상호작용할 수 있는 통합 인터페이스를 제공하는 AI 게이트웨이로, Generative AI 애플리케이션의 제어, 가시성, 보안을 위한 고급 도구를 제공합니다.

> **Langfuse란?** [Langfuse](https://langfuse.com)는 팀이 LLM 호출을 트레이싱하고, 성능을 모니터링하며, AI 애플리케이션의 문제를 디버그하도록 돕는 오픈소스 AI 엔지니어링 플랫폼입니다.

## 3단계: Langfuse OpenAI 드롭인 대체 사용하기

다음으로, Langfuse의 OpenAI 호환 클라이언트(from langfuse.openai import OpenAI)를 사용해 Portkey 게이트웨이를 통해 전송되는 모든 요청을 트레이싱할 수 있습니다. LLM 게이트웨이와 가상 LLM 키의 자세한 설정 방법은 [Portkey 문서](https://portkey.ai/docs/product/ai-gateway)를 참고하세요.

```python
from langfuse.openai import OpenAI
from portkey_ai import createHeaders, PORTKEY_GATEWAY_URL

client = OpenAI(
    api_key="xxx", #Since we are using a virtual key we do not need this
    base_url = PORTKEY_GATEWAY_URL,
    default_headers = createHeaders(
    api_key = "***",
    virtual_key = "***"
    )
)
```

## 4단계: 예제 실행하기

```python
response = client.chat.completions.create(
  model="gpt-4o",  # Or any model supported by your chosen provider
  messages=[
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user", "content": "What are the benefits of using an AI gateway?"},
  ],
)
print(response.choices[0].message.content)

# Flush via global client
langfuse = get_client()
langfuse.flush()
```
