# Langfuse로 Cerebras 옵저버빌리티 구현하기

이 가이드는 [Cerebras](https://cerebras.ai/)를 Langfuse와 통합하는 방법을 보여줍니다. Cerebras의 API는 OpenAI의 클라이언트 라이브러리와 완전히 호환되므로, Langfuse OpenAI 드롭인 대체를 사용하여 애플리케이션의 모든 부분을 트레이싱할 수 있습니다.

> **Cerebras란 무엇인가요?** [Cerebras](https://inference-docs.cerebras.ai/)는 Cerebras의 웨이퍼 스케일 프로세서(WSE)와 CS 시스템을 기반으로 구축된, 대규모 토큰 생성에 특화되어 최적화된 고처리량, 저지연 추론 플랫폼입니다.

> **Langfuse란 무엇인가요?** [Langfuse](https://langfuse.com)는 팀이 API 호출을 트레이싱하고, 성능을 모니터링하고, LLM 애플리케이션의 문제를 디버그하는 데 도움을 주는 오픈 소스 AI 엔지니어링 플랫폼입니다.

## 2단계: 환경 변수 설정

```python
import os

# Get keys for your project from the project settings page: https://cloud.langfuse.com
os.environ.setdefault("LANGFUSE_PUBLIC_KEY", "pk-lf-...")
os.environ.setdefault("LANGFUSE_SECRET_KEY", "sk-lf-...")
os.environ.setdefault("LANGFUSE_BASE_URL", "https://cloud.langfuse.com") # 🇪🇺 EU region
# Other Langfuse data regions include 🇺🇸 US: https://us.cloud.langfuse.com, 🇯🇵 Japan: https://jp.cloud.langfuse.com and ⚕️ HIPAA: https://hipaa.cloud.langfuse.com

# Get your Cerebras API key from https://cloud.cerebras.ai/
os.environ.setdefault("CEREBRAS_API_KEY", "csk-...")
```

## 3단계: Langfuse OpenAI 드롭인 대체

이 단계에서는 `from langfuse.openai import openai`를 임포트하여 네이티브 [OpenAI 드롭인 대체](https://langfuse.com/docs/observability/sdk/instrumentation)를 사용합니다.

Cerebras를 OpenAI의 클라이언트 라이브러리와 함께 사용하려면, `api_key` 옵션에 Cerebras API 키를 전달하고 `base_url`을 `https://api.cerebras.ai/v1`로 변경하세요:

```python
# instead of import openai:
from langfuse.openai import openai

client = openai.OpenAI(
  api_key=os.environ.get("CEREBRAS_API_KEY"),
  base_url="https://api.cerebras.ai/v1",
)
```

## 4단계: 예시 실행하기

다음 셀은 트레이싱된 OpenAI 클라이언트를 사용하여 Cerebras의 채팅 모델을 호출하는 방법을 보여줍니다. 모든 API 호출은 Langfuse에 의해 자동으로 트레이싱됩니다.

```python
response = client.chat.completions.create(
  model="zai-glm-4.6",
  messages=[
    {"role": "system", "content": "You are a travel agent. Be descriptive and helpful."},
    {"role": "user", "content": "Tell me the top 3 things to do in San Francisco"},
  ],
  name="cerebras-travel-agent"
)

print(response.choices[0].message.content)
```

## 5단계: Langfuse에서 트레이스 확인하기

예시 모델 호출을 실행한 후 Langfuse에서 트레이스를 확인할 수 있습니다. Cerebras API 호출에 대한 다음과 같은 상세 정보를 확인할 수 있습니다:

- 요청 매개변수(모델, 메시지, temperature 등)
- 응답 콘텐츠
- 토큰 사용량 통계
- 지연 시간 메트릭
