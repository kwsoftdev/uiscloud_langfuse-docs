# Langfuse를 LlamaIndex와 통합하기

이 노트북은 LLM 애플리케이션의 관찰 가능성과 디버깅을 위해 **Langfuse**를 **LlamaIndex**와 통합하는 방법을 단계별로 안내하는 가이드입니다.

> **LlamaIndex란 무엇인가요?** [LlamaIndex](https://www.llamaindex.ai/) ([GitHub](https://github.com/run-llama/llama_index))는 프라이빗 데이터로 LLM을 보강하기 위해 맞춤 설계된 고급 "데이터 프레임워크"입니다. 다재다능한 데이터 커넥터를 통해 다양한 데이터 소스와 형식(API, PDF, 문서, SQL 등)의 통합을 간소화하고, LLM 호환성을 위해 데이터를 인덱스와 그래프로 구조화합니다. 이 플랫폼은 컨텍스트에 특화된 출력으로 LLM 입력을 보강하기 위한 정교한 검색/쿼리 인터페이스를 제공합니다.

> **Langfuse란 무엇인가요?** [Langfuse](https://langfuse.com)는 오픈소스 AI 엔지니어링 플랫폼입니다. LLM 애플리케이션을 위한 트레이싱 및 모니터링 기능을 제공합니다. Langfuse는 네이티브 통합, OpenTelemetry, 전용 SDK를 통해 다양한 도구 및 프레임워크와 통합하여 상세한 인사이트를 제공함으로써 개발자가 AI 시스템을 디버그, 분석, 최적화하도록 돕습니다.

## 시작하기

포괄적인 트레이싱을 위해 LlamaIndex를 사용하고 이를 Langfuse와 통합하는 실용적인 예제를 살펴보겠습니다.

<Steps>
### 1단계: 의존성 설치하기

```python
%pip install langfuse openinference-instrumentation-llama-index llama-index-llms-openai llama-index -U
```

### 2단계: Langfuse SDK 구성하기

다음으로, Langfuse API 키를 설정하세요. 무료 [Langfuse Cloud](https://langfuse.com/cloud) 계정에 가입하거나 [Langfuse를 셀프 호스팅](https://langfuse.com/self-hosting)하여 이 키를 받을 수 있습니다. 이 환경 변수들은 Langfuse 클라이언트가 인증하고 Langfuse 프로젝트로 데이터를 전송하는 데 필수적입니다.

```python
import os

# Get keys for your project from the project settings page: https://cloud.langfuse.com

os.environ.setdefault("LANGFUSE_PUBLIC_KEY", "pk-lf-...")
os.environ.setdefault("LANGFUSE_SECRET_KEY", "sk-lf-...")
os.environ.setdefault("LANGFUSE_BASE_URL", "https://cloud.langfuse.com") # 🇪🇺 EU region
# Other Langfuse data regions include 🇺🇸 US: https://us.cloud.langfuse.com, 🇯🇵 Japan: https://jp.cloud.langfuse.com and ⚕️ HIPAA: https://hipaa.cloud.langfuse.com

# Your OpenAI key
os.environ.setdefault("OPENAI_API_KEY", "sk-proj-...")

```

환경 변수가 설정되면 이제 Langfuse 클라이언트를 초기화할 수 있습니다. `get_client()`는 환경 변수로 제공된 자격 증명을 사용해 Langfuse 클라이언트를 초기화합니다.

```python
from langfuse import get_client

langfuse = get_client()

# Verify connection
if langfuse.auth_check():
    print("Langfuse client is authenticated and ready!")
else:
    print("Authentication failed. Please check your credentials and host.")

```

### 3단계: LlamaIndex 계측 초기화하기

이제 [OpenInference LlamaIndex 계측](https://docs.arize.com/phoenix/tracing/integrations-tracing/llamaindex)을 초기화합니다. 이 서드파티 계측은 LlamaIndex 작업을 자동으로 캡처하고 OpenTelemetry(OTel) span을 Langfuse로 내보냅니다.

```python
from openinference.instrumentation.llama_index import LlamaIndexInstrumentor

# Initialize LlamaIndex instrumentation
LlamaIndexInstrumentor().instrument()
```

### 4단계: 기본 LlamaIndex 애플리케이션

간단한 LlamaIndex 애플리케이션을 만들어 보겠습니다. 이 예제에서는 질문에 답할 수 있는 간단한 쿼리 엔진을 만듭니다. 이는 Langfuse 트레이싱을 보여주기 위한 기반이 될 것입니다.

```python
from llama_index.llms.openai import OpenAI

llm = OpenAI(model="gpt-4o")

with langfuse.start_as_current_observation(as_type="span", name="llama-index-trace"):
    response = llm.complete("Hello, world!")
    print(response)

langfuse.flush()
```

### 5단계: Langfuse에서 트레이스 확인하기

애플리케이션을 실행한 후 Langfuse Trace Table로 이동하세요. LLM 호출, 검색 작업, 입력, 출력, 성능 메트릭에 대한 인사이트를 제공하는 애플리케이션 실행의 상세한 트레이스를 확인할 수 있습니다. 트레이스는 쿼리 처리부터 문서 검색을 거쳐 응답 생성에 이르는 전체 흐름을 보여줍니다.
