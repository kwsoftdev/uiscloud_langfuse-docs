# Langfuse로 Mastra 관찰 가능성 확보하기

이 가이드는 관찰 가능성과 트레이싱을 위해 **Langfuse**를 **Mastra**와 통합하는 방법을 보여줍니다. 이 단계를 따르면 Langfuse 대시보드에서 Mastra 에이전트를 모니터링하고 디버그할 수 있게 됩니다.

> **Mastra란 무엇인가요?** [Mastra](https://mastra.ai)는 AI 애플리케이션 구축에 필요한 핵심 요소를 제공하도록 설계된 TypeScript 에이전트 프레임워크입니다. 개발자가 메모리와 도구 호출 기능을 갖춘 AI 에이전트를 만들고, 결정론적 LLM 워크플로우를 구현하며, 지식 통합을 위해 RAG를 활용할 수 있게 해줍니다.

> **Langfuse란 무엇인가요?** [Langfuse](https://langfuse.com)는 오픈소스 AI 엔지니어링 플랫폼입니다. LLM 애플리케이션을 위한 트레이싱 및 모니터링 기능을 제공합니다.

## 통합

<Steps>

### Mastra 프로젝트 생성하기

아직 Mastra 프로젝트가 없다면 Mastra CLI를 사용해 하나 만들 수 있습니다:

```bash
npx create-mastra
```

프로젝트 디렉터리로 이동하세요:

```bash
cd your-mastra-project
```

전체 Mastra 설치 방법은 [여기](https://mastra.ai/docs/getting-started/installation)에서 확인할 수 있습니다

### Langfuse 프로젝트 설정하기

[Langfuse](https://cloud.langfuse.com)에서 프로젝트를 생성하고 프로젝트 설정 페이지에서 API 키를 가져오세요.

### 환경 변수 추가하기

`.env.development`(또는 `.env`) 파일을 다음 변수로 생성하거나 업데이트하세요:

```bash
# Your LLM API key
OPENAI_API_KEY=your-api-key

# Langfuse credentials
LANGFUSE_SECRET_KEY=sk-lf-...
LANGFUSE_PUBLIC_KEY=pk-lf-...
LANGFUSE_BASE_URL=https://cloud.langfuse.com # Optional. Defaults to https://cloud.langfuse.com
```

### `@mastra/langfuse` 패키지 설치하기

프로젝트에 `@mastra/langfuse` 패키지를 추가하세요:

```bash
npm install @mastra/langfuse
```

### 에이전트 설정하기

프로젝트에 에이전트를 만드세요. 예를 들어, `agents/chefAgent.ts` 파일을 만듭니다:

```typescript
import { Agent } from "@mastra/core/agent";
import { openai } from "@ai-sdk/openai";

export const chefAgent = new Agent({
  name: "chef-agent",
  instructions:
    "You are Michel, a practical and experienced home chef " +
    "You help people cook with whatever ingredients they have available.",
  model: openai("gpt-4o-mini"),
});
```

{" "}

<Callout type="info">`ai-sdk`에서 어떤 모델 제공업체든 사용할 수 있습니다.</Callout>

### 에이전트 등록 및 Langfuse 구성하기

에이전트를 등록하고 Langfuse 통합을 구성하기 위해 Mastra 인스턴스 구성을 만들거나 업데이트하세요. 예를 들어, `mastra.ts` 파일을 만듭니다:

```typescript
import { Mastra } from "@mastra/core";
import { LangfuseExporter } from "@mastra/langfuse";
import { chefAgent } from "./agents/chefAgent";

export const mastra = new Mastra({
  agents: { chefAgent },
  observability: {
    configs: {
      langfuse: {
        serviceName: "my-service",
        exporters: [
          new LangfuseExporter({
            publicKey: process.env.LANGFUSE_PUBLIC_KEY!,
            secretKey: process.env.LANGFUSE_SECRET_KEY!,
            baseUrl: process.env.LANGFUSE_BASE_URL,
            options: {
              environment: process.env.NODE_ENV,
            },
          }),
        ],
      },
    },
  },
});
```

### mastra dev 서버 실행하기

Mastra 개발 서버를 시작하세요:

```bash
npm run dev
```

제공된 URL로 개발자 플레이그라운드로 이동하여 에이전트와 채팅을 시작하세요.

### Langfuse에서 트레이스 확인하기

[Langfuse 대시보드](https://cloud.langfuse.com)로 이동하면 에이전트 상호작용의 트레이스를 확인할 수 있습니다. AI 상호작용의 프롬프트, 완료 내용, 기타 세부 정보를 분석할 수 있습니다.

트레이스 예제입니다:

![Langfuse UI의 Mastra 트레이스](/images/docs/langfuse-mastra-trace.png)

</Steps>

## 구성 옵션

### 실시간 모드 vs 배치 모드

Langfuse 익스포터는 트레이스를 전송하는 두 가지 모드를 지원합니다:

- **실시간(개발용)**: 즉각적인 가시성을 위해 각 이벤트마다 플러시합니다.

```ts
new LangfuseExporter({
  publicKey: process.env.LANGFUSE_PUBLIC_KEY!,
  secretKey: process.env.LANGFUSE_SECRET_KEY!,
  realtime: true, // Flush after each event
})
```

- **배치(프로덕션용, 기본값)**: 더 나은 처리량을 위해 버퍼링하여 배치로 전송합니다.

```ts
new LangfuseExporter({
  publicKey: process.env.LANGFUSE_PUBLIC_KEY!,
  secretKey: process.env.LANGFUSE_SECRET_KEY!,
  realtime: false, // Default behavior
})
```

### 전체 구성

```ts
new LangfuseExporter({
  // Required credentials
  publicKey: process.env.LANGFUSE_PUBLIC_KEY!,
  secretKey: process.env.LANGFUSE_SECRET_KEY!,

  // Optional settings
  baseUrl: process.env.LANGFUSE_BASE_URL, // Defaults to https://cloud.langfuse.com
  realtime: process.env.NODE_ENV === "development",
  logLevel: "info", // debug | info | warn | error

  // Langfuse-specific options
  options: {
    environment: process.env.NODE_ENV,
    version: process.env.APP_VERSION,
    release: process.env.GIT_COMMIT,
  },
})
```

## 문제 해결

- **NextJS 통합 문제**: NextJS 애플리케이션에서 Mastra를 Langfuse와 함께 사용할 때 문제가 발생하면, NextJS 관련 구성 및 설정 방법에 대해 [Mastra NextJS 트레이싱 문서](https://mastra.ai/en/docs/observability/nextjs-tracing)를 참고하세요.
- **Langfuse에 트레이스가 없는 경우**: 자격 증명이 올바른지 확인하고 이 [문제 해결 가이드](/faq/all/missing-traces)를 따르세요
