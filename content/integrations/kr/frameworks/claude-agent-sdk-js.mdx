# Langfuse로 Claude Agent SDK JS/TS 트레이싱하기

<a href="https://langfuse.com/integrations/frameworks/claude-agent-sdk"><img className="inline" alt="Python" src="https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white" /></a> <a href="https://langfuse.com/integrations/frameworks/claude-agent-sdk-js"><img className="inline" alt="JS/TS" src="https://img.shields.io/badge/JS/TS-d4d4d8?style=flat&logo=javascript&logoColor=white" /></a>

[Claude Agent SDK](https://github.com/anthropics/claude-agent-sdk-typescript)는 TypeScript로 AI 에이전트를 구축하기 위한 Anthropic의 오픈 소스 프레임워크입니다. Claude Agent SDK를 **Langfuse**와 결합하면 개발 및 프로덕션 환경에서 모든 에이전트 단계, 도구 호출, 모델 응답을 트레이싱, 모니터링, 분석할 수 있습니다.

이 노트북은 [OpenInference](https://github.com/Arize-ai/openinference)의 [`ClaudeAgentSDKInstrumentation`](https://github.com/Arize-ai/openinference/tree/main/js/packages/openinference-instrumentation-claude-agent-sdk) 라이브러리를 사용하여 Claude Agent SDK 호출을 자동으로 계측하고 OpenTelemetry 스팬을 Langfuse로 전송하는 방법을 보여줍니다.

> **Claude Agent SDK란 무엇인가요?**\
> [Claude Agent SDK](https://docs.anthropic.com/en/docs/claude-code/sdk/sdk-typescript)는 AI 에이전트를 구축하기 위한 Anthropic의 오픈 소스 프레임워크입니다. MCP 서버 및 커스텀 도구에 대한 네이티브 지원을 포함하여 `query()` 함수로 도구를 사용하는 에이전트를 만들기 위한 깔끔한 API를 제공합니다.

> **Langfuse란 무엇인가요?**\
> [Langfuse](https://langfuse.com)는 LLM 관찰 가능성 및 모니터링을 위한 오픈 소스 플랫폼입니다. 메타데이터, 프롬프트 상세 정보, 토큰 사용량, 지연 시간 등을 캡처하여 AI 애플리케이션을 트레이싱하고 모니터링하는 데 도움을 줍니다.

<Steps>
## 1단계: 의존성 설치

필요한 패키지를 설치합니다:

```bash
npm install @anthropic-ai/claude-agent-sdk @arizeai/openinference-instrumentation-claude-agent-sdk @langfuse/otel @opentelemetry/sdk-node
```

> **참고**: 이 쿡북은 실행을 위해 **Deno.js**를 사용하며, 패키지 임포트와 환경 변수 설정에 다른 구문이 필요합니다. Node.js 애플리케이션의 경우 설정 과정은 유사하지만 표준 `npm` 패키지와 `process.env`를 사용합니다.

## 2단계: 환경 구성

Langfuse와 Anthropic API 키를 설정합니다. Langfuse 키는 무료 [Langfuse Cloud](https://langfuse.com/cloud) 계정에 가입하거나 [Langfuse를 셀프 호스팅](https://langfuse.com/self-hosting)하여 얻을 수 있습니다. Anthropic API 키는 [Anthropic Console](https://console.anthropic.com/)에서 얻으세요.

```typescript
Deno.env.set("ANTHROPIC_API_KEY", "sk-ant-...");

Deno.env.set("LANGFUSE_PUBLIC_KEY", "pk-lf-...");
Deno.env.set("LANGFUSE_SECRET_KEY", "sk-lf-...");

Deno.env.set("LANGFUSE_BASE_URL", "https://cloud.langfuse.com"); // 🇪🇺 EU region
// Other Langfuse data regions include 🇺🇸 US: https://us.cloud.langfuse.com, 🇯🇵 Japan: https://jp.cloud.langfuse.com and ⚕️ HIPAA: https://hipaa.cloud.langfuse.com
```

## 3단계: Langfuse로 OpenTelemetry 초기화

`LangfuseSpanProcessor`와 OpenInference의 [`ClaudeAgentSDKInstrumentation`](https://github.com/Arize-ai/openinference/tree/main/js/packages/openinference-instrumentation-claude-agent-sdk)로 OpenTelemetry SDK를 설정합니다. 이 인스트루멘테이션은 에이전트 실행 및 도구 호출을 자동으로 캡처하여 OpenTelemetry 스팬으로 Langfuse에 전송합니다.

또한 기본 Langfuse 필터와 함께 `@arizeai/openinference-instrumentation-claude-agent-sdk` 인스트루멘테이션 스코프의 스팬을 포함하기 위한 커스텀 [`shouldExportSpan`](https://langfuse.com/docs/observability/sdk/advanced-features#filtering-by-instrumentation-scope) 필터를 제공합니다.

```typescript
import { NodeSDK } from "npm:@opentelemetry/sdk-node";
import { LangfuseSpanProcessor, isDefaultExportSpan } from "npm:@langfuse/otel";
import { ClaudeAgentSDKInstrumentation } from "npm:@arizeai/openinference-instrumentation-claude-agent-sdk";

import * as ClaudeAgentSDKModule from "npm:@anthropic-ai/claude-agent-sdk";

// Create mutable copy — Deno's ESM namespace objects are read-only
const ClaudeAgentSDK = { ...ClaudeAgentSDKModule };

const instrumentation = new ClaudeAgentSDKInstrumentation();
instrumentation.manuallyInstrument(ClaudeAgentSDK);

const sdk = new NodeSDK({
  spanProcessors: [
    new LangfuseSpanProcessor({
      shouldExportSpan: ({ otelSpan }) =>
        isDefaultExportSpan(otelSpan) ||
        otelSpan.instrumentationScope.name ===
          "@arizeai/openinference-instrumentation-claude-agent-sdk",
    }),
  ],
  instrumentations: [instrumentation],
});

sdk.start();
```

## 4단계: 에이전트 실행

Claude Agent SDK의 `query()` 함수를 사용하여 에이전트를 실행합니다. 모든 에이전트 단계, 도구 호출, 모델 완료는 자동으로 트레이싱되어 Langfuse로 전송됩니다.

```typescript
const { query } = ClaudeAgentSDK;

for await (const message of query({
  prompt: "What is the capital of France? Answer in a single sentence.",
  options: {
    model: "claude-sonnet-4-5",
  },
})) {
  if (message.type === "assistant") {
    console.log(message.message.content);
  }
}

await sdk.shutdown();
```
