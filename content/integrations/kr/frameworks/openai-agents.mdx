# Langfuse로 OpenAI Agents SDK 트레이싱하기

이 노트북은 AI 에이전트를 모니터링, 디버그, 평가하기 위해 **OpenAI Agents** 워크플로우에 **Langfuse를 통합**하는 방법을 보여줍니다.

> **OpenAI Agents SDK란 무엇인가요?**: [OpenAI Agents SDK](https://openai.github.io/openai-agents-python/)는 개발자가 AI 에이전트를 구축하고 멀티 에이전트 워크플로우를 조율할 수 있게 해주는 가벼운 오픈소스 프레임워크입니다. 도구, 핸드오프, 가드레일과 같은 구성 요소를 제공하여 커스텀 지침과 통합 도구로 대규모 언어 모델을 설정할 수 있습니다. Python 우선 설계는 동적 지침과 함수 도구를 지원하여 빠른 프로토타이핑과 외부 시스템과의 통합을 돕습니다.

> **Langfuse란 무엇인가요?**: [Langfuse](https://langfuse.com/)는 AI 에이전트를 위한 오픈소스 관찰 가능성 플랫폼입니다. LLM 호출, 도구 사용, 비용, 지연 시간 등을 시각화하고 모니터링하는 데 도움을 줍니다.

## 3. 에이전트 계측하기

이제 Langfuse 클라이언트를 초기화합니다. `get_client()`는 환경 변수로 제공된 자격 증명을 사용해 Langfuse 클라이언트를 초기화합니다.

```python

langfuse = get_client()

# Verify connection
if langfuse.auth_check():
    print("Langfuse client is authenticated and ready!")
else:
    print("Authentication failed. Please check your credentials and host.")
```

## 4. Hello World 예제

아래에서는 항상 **하이쿠** 형식으로 답하는 **OpenAI 에이전트**를 만듭니다. `Runner.run`으로 이를 실행하고 최종 출력을 출력합니다.

```python
import asyncio
from agents import Agent, Runner

async def main():
    agent = Agent(
        name="Assistant",
        instructions="You only respond in haikus.",
    )

    result = await Runner.run(agent, "Tell me about recursion in programming.")
    print(result.final_output)

loop = asyncio.get_running_loop()
await loop.create_task(main())
```

위의 링크(또는 자신의 프로젝트 링크)를 클릭하면 디버깅이나 최적화를 위해 모든 하위 span, 토큰 사용량, 지연 시간 등을 볼 수 있습니다.

## 5. 멀티 에이전트 핸드오프 예제

여기서는 다음을 만듭니다:

- 스페인어로만 응답하는 **스페인어 에이전트**.
- 영어로만 응답하는 **영어 에이전트**.
- 입력 언어에 따라 요청을 올바른 에이전트로 라우팅하는 **분류(triage) 에이전트**.

모든 호출이나 핸드오프는 트레이스의 일부로 캡처됩니다. 이를 통해 어떤 하위 에이전트나 도구가 사용되었는지, 그리고 최종 결과가 무엇인지 확인할 수 있습니다.

```python
from agents import Agent, Runner
import asyncio

spanish_agent = Agent(
    name="Spanish agent",
    instructions="You only speak Spanish.",
)

english_agent = Agent(
    name="English agent",
    instructions="You only speak English",
)

triage_agent = Agent(
    name="Triage agent",
    instructions="Handoff to the appropriate agent based on the language of the request.",
    handoffs=[spanish_agent, english_agent],
)

result = await Runner.run(triage_agent, input="Hola, ¿cómo estás?")
print(result.final_output)
```

## 6. 함수 예제

OpenAI Agents SDK는 에이전트가 Python 함수를 호출할 수 있게 해줍니다. Langfuse 계측을 사용하면 어떤 **함수**가 호출되었는지, 그 인자와 반환값이 무엇인지 확인할 수 있습니다. 여기서는 간단한 함수 `get_weather(city: str)`를 정의하고 도구로 추가합니다.

```python
import asyncio
from agents import Agent, Runner, function_tool

# Example function tool.
@function_tool
def get_weather(city: str) -> str:
    return f"The weather in {city} is sunny."

agent = Agent(
    name="Hello world",
    instructions="You are a helpful agent.",
    tools=[get_weather],
)

async def main():
    result = await Runner.run(agent, input="What's the weather in Tokyo?")
    print(result.final_output)

loop = asyncio.get_running_loop()
await loop.create_task(main())
```

![Langfuse의 예제 트레이스](https://langfuse.com/images/cookbook/integration_openai-agents/openai-agent-example-trace-function.png)

트레이스를 볼 때, 함수 호출 `get_weather`와 전달된 인자를 캡처하는 span을 볼 수 있습니다.

## 7. 에이전트 실행 그룹화하기

일부 워크플로우에서는 여러 호출을 하나의 트레이스로 그룹화하고 싶을 수 있습니다. 예를 들어, 동일한 사용자 요청과 관련된 작은 프롬프트 체인을 구축하는 경우입니다. `trace(...)` 컨텍스트 매니저를 사용하여 여러 호출을 하나의 최상위 트레이스 아래에 중첩할 수 있습니다.

```python
from agents import Agent, Runner, trace
import asyncio

async def main():
    agent = Agent(name="Joke generator", instructions="Tell funny jokes.")

    with trace("Joke workflow"):
        first_result = await Runner.run(agent, "Tell me a joke")
        second_result = await Runner.run(agent, f"Rate this joke: {first_result.final_output}")
        print(f"Joke: {first_result.final_output}")
        print(f"Rating: {second_result.final_output}")

loop = asyncio.get_running_loop()
await loop.create_task(main())
```

![Langfuse의 예제 트레이스](https://langfuse.com/images/cookbook/integration_openai-agents/openai-agent-example-trace-grouped.png)

**예제**: [Langfuse 트레이스](https://cloud.langfuse.com/project/cloramnkj0002jz088vzn1ja4/traces/afa1ae379b6c3a5333e2bab357f31cdc?timestamp=2025-09-30T09%3A04%3A06.678Z&observation=496efdc060a5399b)

각 하위 호출은 최상위 **Joke workflow** span 아래의 하위 span으로 표현되어, 전체 대화나 호출 순서를 쉽게 확인할 수 있습니다.

## Langfuse 프롬프트 연결하기

[Langfuse Prompt Management](https://langfuse.com/docs/prompt-management/overview)로 프롬프트를 관리하는 경우, OTel Span 프로세서를 설정하여 사용된 프롬프트를 트레이스에 연결할 수 있습니다.

<Callout type="info" emoji="⚠️">
**제한 사항:** 이 방법은 정의된 문자열로 시작하는 트레이스 내 모든 generation span에 Langfuse 프롬프트를 연결합니다(다음 셀 참고).

</Callout>

```python
from contextvars import ContextVar
from typing import Optional
from opentelemetry import context as context_api, trace
from opentelemetry.sdk.trace import TracerProvider
from opentelemetry.sdk.trace.export import Span, SpanProcessor

prompt_info_var = ContextVar("prompt_info", default=None)

# Make sure to set the name of the generation spans in your trace
class LangfuseProcessor(SpanProcessor):
    def on_start(self, span: "Span", parent_context: Optional[context_api.Context] = None) -> None:
        if span.name.startswith("response"): # The name of the generation spans in your trace
            prompt_info = prompt_info_var.get()
            if prompt_info:
                span.set_attribute("langfuse.prompt.name", prompt_info.get("name"))
                span.set_attribute("langfuse.prompt.version", prompt_info.get("version"))


from langfuse import get_client
langfuse = get_client()

trace.get_tracer_provider().add_span_processor(LangfuseProcessor())
```

```python
from openinference.instrumentation.openai_agents import OpenAIAgentsInstrumentor

OpenAIAgentsInstrumentor().instrument()
```

```python
# Fetch the prompt from Langfuse Prompt Management
langfuse_prompt = langfuse.get_prompt("movie-critic")

# Pass the prompt to the SpanProcessor
prompt_info_var.set({
    "name": langfuse_prompt.name,
    "version": langfuse_prompt.version,
})

# Run the agent ...
```
