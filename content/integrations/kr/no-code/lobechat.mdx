# Langfuse Integration with LobeChat

## LobeChat이란 무엇인가요?

[LobeChat](https://lobehub.com/)은 다양한 AI 모델과 도구를 매끄럽게 통합하는 **[오픈소스](https://github.com/lobehub/lobe-chat) LLM 채팅 플랫폼**으로, 사용자에게 고급 언어 기술과 상호작용할 수 있는 직관적인 인터페이스를 제공합니다.

## Langfuse란 무엇인가요?

[Langfuse](https://langfuse.com/)는 가장 많이 사용되는 **오픈소스 LLM 관측 가능성 플랫폼** 중 하나입니다. Langfuse 인테그레이션을 활성화하면, 애플리케이션 데이터를 Langfuse로 추적해 LobeChat의 사용을 개발, 모니터링, 개선할 수 있습니다:

- 애플리케이션 [트레이스](/docs/observability/overview)
- 사용 패턴
- 사용자 및 모델별 비용 데이터
- [평가](/docs/evaluation/overview)

## 시작하기

<Steps>

### Langfuse 설정하기

[Langfuse Cloud](https://cloud.langfuse.com)에 가입하거나 Langfuse를 [셀프 호스팅](/self-hosting)해서 Langfuse API 키를 받으세요.

### LobeChat 설정하기

[LobeChat을 셀프 호스팅](https://lobehub.com/docs/self-hosting/start)하는 방법은 여러 가지가 있습니다. 이 예시에서는 Docker Desktop 배포를 사용합니다.

<Tabs items={["Environment Variables", "Example in Docker Desktop"]}>
<Tab>

LobeChat을 배포하기 전에, 이전 단계에서 만든 Langfuse API 키로 다음 네 개의 환경 변수를 설정하세요.

```sh
ENABLE_LANGFUSE = '1'
LANGFUSE_SECRET_KEY = 'sk-lf...'
LANGFUSE_PUBLIC_KEY = 'pk-lf...'
LANGFUSE_HOST = 'https://cloud.langfuse.com'
```

</Tab>

<Tab>

Docker 컨테이너를 실행하기 전에, 이전 단계에서 만든 Langfuse API 키로 Docker Desktop에서 환경 변수를 설정하세요.

<Frame className="sm:w-1/2">
![Environment Variables in Docker Desktop](/images/docs/lobechat-docker-desktop-env.png)
</Frame>
</Tab>

</Tabs>

### 설정에서 애널리틱스 활성화하기

LobeChat이 실행되면, **Settings**의 **About** 탭으로 이동해 애널리틱스를 활성화하세요. 트레이스가 Langfuse로 전송되려면 이 작업이 필요합니다.

<Frame className="mt-5">
  ![LobeChat Settings](/images/docs/lobechat-settings.png)
</Frame>

### Langfuse에서 트레이스 확인하기

LLM 모델 키를 설정한 뒤, LobeChat 애플리케이션과 상호작용을 시작할 수 있습니다.

<Frame className="mt-5">
  ![LobeChat Conversation](/images/docs/lobechat-converstation.png)
</Frame>

채팅의 모든 대화는 자동으로 추적되어 Langfuse로 전송됩니다. Langfuse 플랫폼의 [Traces 섹션](/docs/observability/overview)에서 트레이스를 확인할 수 있습니다.

<Frame className="mt-5">
  ![LobeChat Example Trace](/images/docs/lobechat-example-trace.png)
</Frame>
