# Integrating Goose with Langfuse

[Goose](https://github.com/block/goose)는 엔지니어링 작업을 매끄럽게 자동화하는 오픈소스 온머신 AI 에이전트입니다. Goose를 [Langfuse](https://langfuse.com)와 통합하면 Goose 요청을 모니터링하고 에이전트가 어떻게 동작하는지 파악할 수 있습니다.

## Goose 소개

<Video
  src="https://static.langfuse.com/docs-videos/goose_hero.mp4"
  aspectRatio={2702 / 1696}
  gifStyle
/>

Goose는 여러분의 머신에서 로컬로 실행되며 일반적인 엔지니어링 워크플로우를 자동화하는 강력한 오픈소스 AI 에이전트입니다. 개발자가 반복적인 작업에 시간을 덜 쓰고 더 많은 시간을 만들기에 쓸 수 있도록 돕기 위해 설계되었습니다.

핵심 기능:

- 완전히 오픈소스이며 확장 가능해, 특정 요구에 맞게 Goose를 커스터마이즈할 수 있습니다
- 실행과 데이터에 대한 제어권을 유지시켜 주는 로컬 우선 아키텍처
- 선호하는 LLM 제공업체와 Model Context Protocol(MCP) 서버와의 유연한 통합 옵션
- 최소한의 감독으로 복잡한 엔지니어링 작업을 처리하는 내장 지능

다음은 개발자들이 Goose를 사용하는 실제 사례입니다:

- 복잡한 API 비즈니스 로직을 정확하게 반영하는 포괄적인 테스트 데이터 생성
- 팀 조율을 간소화하기 위한 Google Workspace 자동화 스크립트 구축
- React 컴포넌트 업데이트와 같은 대규모 코드베이스 마이그레이션 실행
- 개발 워크플로우를 최적화하기 위한 커스텀 CLI 도구 생성 및 관리

GitHub: [https://github.com/block/goose](https://github.com/block/goose)

## 시작하기

이 가이드는 Goose를 Langfuse와 통합하는 단계를 안내합니다.

<Steps>

### Langfuse 설정하기

Langfuse API 키를 받으려면 [여기](https://cloud.langfuse.com)에서 Langfuse Cloud에 가입하거나 [Docker Compose](https://langfuse.com/self-hosting/deployment/docker-compose)로 Langfuse를 셀프 호스팅하세요.

### Langfuse에 연결하도록 Goose 구성하기

Goose(Rust로 작성됨)가 Langfuse 서버에 연결할 수 있도록 환경 변수를 설정하세요.

```bash
export LANGFUSE_INIT_PROJECT_PUBLIC_KEY=pk-lf-...
export LANGFUSE_INIT_PROJECT_SECRET_KEY=sk-lf-...
export LANGFUSE_HOST=https://cloud.langfuse.com # EU data region 🇪🇺

# Other Langfuse data regions: https://us.cloud.langfuse.com (🇺🇸 US), https://jp.cloud.langfuse.com (🇯🇵 Japan), https://hipaa.cloud.langfuse.com (⚕️ HIPAA)
# https://localhost:3000 if you're self-hosting
```

### Langfuse 인테그레이션과 함께 Goose 실행하기

이제 Goose를 실행하고 Langfuse를 통해 AI 요청과 동작을 모니터링할 수 있습니다.

Goose가 실행 중이고 환경 변수가 설정되어 있으면, Langfuse가 Goose 활동의 트레이스를 캡처하기 시작합니다.

_[Langfuse의 예시 트레이스(공개)](https://cloud.langfuse.com/project/cloramnkj0002jz088vzn1ja4/traces/cea4ed38-0c44-4b0a-8c20-4b0b6b9e8d73?timestamp=2025-01-31T15%3A52%3A30.362Z&observation=7c8e5807-3c29-4c28-9c6f-7d7427be401f)_
