# Trace LibreChat with Langfuse

이 가이드에서는 [Langfuse](https://langfuse.com) 트레이싱을 [LibreChat](https://www.librechat.ai/)과 통합하여 AI 대화에 대한 완전한 옵저버빌리티를 얻는 방법을 보여드립니다.

> **LibreChat이란 무엇인가요?** [LibreChat](https://github.com/danny-avila/LibreChat)은 여러 AI 제공업체(OpenAI, Anthropic, Google, Azure 등)를 하나의 통합된 인터페이스로 모은 오픈 소스 AI 채팅 플랫폼입니다.

> **Langfuse란 무엇인가요?** [Langfuse](https://github.com/langfuse/langfuse)는 LLM 애플리케이션을 추적, 모니터링, 디버깅할 수 있도록 돕는 오픈 소스 LLM 옵저버빌리티 플랫폼입니다.

<Steps>
## 사전 요구 사항

시작하기 전에 다음을 준비하세요.

1. 실행 중인 LibreChat 인스턴스([LibreChat 설치 가이드](https://www.librechat.ai/docs/quick_start/local_setup) 참고)
2. Langfuse 계정([무료로 가입하기](https://cloud.langfuse.com))
3. 프로젝트 설정에서 발급받은 Langfuse API 키

## LibreChat 환경 변수 구성하기

LibreChat은 구성을 위해 환경 변수를 사용합니다. LibreChat 설치 디렉터리의 `.env` 파일에 다음 Langfuse 관련 변수를 추가하세요.

```bash
# Langfuse Configuration
LANGFUSE_PUBLIC_KEY=pk-lf-***
LANGFUSE_SECRET_KEY=sk-lf-***

# 🇪🇺 EU Data Region
LANGFUSE_BASE_URL=https://cloud.langfuse.com

# 🇺🇸 US Data Region
# Other Langfuse data regions include 🇺🇸 US: https://us.cloud.langfuse.com, 🇯🇵 Japan: https://jp.cloud.langfuse.com and ⚕️ HIPAA: https://hipaa.cloud.langfuse.com
```

> **참고**: 셀프 호스팅 Langfuse 인스턴스의 경우 `LANGFUSE_BASE_URL`을 사용자 지정 URL(예: 로컬 개발 환경의 경우 `http://localhost:3000`)로 설정하세요.

## LibreChat 재시작하기

환경 변수를 추가한 후 LibreChat 인스턴스를 재시작하여 변경 사항을 적용하세요.

## Langfuse에서 트레이스 확인하기

Langfuse가 구성된 상태로 LibreChat이 재시작되면, Langfuse UI에서 모든 채팅 메시지 응답에 대한 새 트레이스를 확인할 수 있습니다.
