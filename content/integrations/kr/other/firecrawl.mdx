# Firecrawl Integration

이 가이드에서는 [Langfuse](https://langfuse.com)를 [Firecrawl](https://www.firecrawl.dev/)과 통합하여 웹 스크래핑 작업을 추적하는 방법을 보여드립니다. Langfuse의 트레이싱 기능을 활용하면 Firecrawl 함수의 입력, 출력, 실행 시간과 같은 세부 정보를 자동으로 캡처할 수 있습니다.

> **Firecrawl이란 무엇인가요?** [Firecrawl](https://www.firecrawl.dev/)은 웹사이트를 크롤링하는 API 서비스로, 대상 URL과 접근 가능한 하위 페이지를 자동으로 처리하고 가져온 콘텐츠를 깔끔하고 LLM에 바로 사용할 수 있는 markdown으로 변환합니다. HTML, 메타데이터 등을 포함한 웹 콘텐츠를 다양한 애플리케이션을 위한 구조화된 데이터로 변환하는 강력한 스크래핑, 크롤링, 추출 기능을 제공합니다.

> **Langfuse란 무엇인가요?** [Langfuse](https://langfuse.com)는 팀이 LLM 애플리케이션에서 API 호출을 추적하고, 성능을 모니터링하고, 문제를 디버깅할 수 있도록 돕는 오픈 소스 AI 엔지니어링 플랫폼입니다.

## 시작하기

먼저 필요한 Python 패키지를 설치합니다.

```python
%pip install langfuse firecrawl-py
```

다음으로 Langfuse API 키로 환경을 구성합니다. 키는 Langfuse 대시보드에서 얻을 수 있습니다.

```python
import os
# Get keys for your project from the project settings page https://cloud.langfuse.com

os.environ.setdefault("LANGFUSE_PUBLIC_KEY", "pk-lf-...")
os.environ.setdefault("LANGFUSE_SECRET_KEY", "sk-lf-...")
os.environ.setdefault("LANGFUSE_BASE_URL", "https://cloud.langfuse.com") # 🇪🇺 EU region
# Other Langfuse data regions include 🇺🇸 US: https://us.cloud.langfuse.com, 🇯🇵 Japan: https://jp.cloud.langfuse.com and ⚕️ HIPAA: https://hipaa.cloud.langfuse.com
```

Firecrawl API 키로 Firecrawl 애플리케이션을 초기화합니다.

```python
from firecrawl import FirecrawlApp

app = FirecrawlApp(api_key="fc-...")
```

Firecrawl 스크래퍼를 모니터링하기 위해 [Langfuse `@observe()` 데코레이터](https://langfuse.com/docs/sdk/python/decorators)를 사용합니다. 이 예제에서 `@observe()` 데코레이터는 `scrape_website()` 함수의 입력, 출력, 실행 시간을 캡처합니다. 모든 트레이스 데이터는 자동으로 Langfuse에 전송되어 Firecrawl 작업을 실시간으로 모니터링할 수 있습니다.

```python
from langfuse import observe

@observe()
def scrape_website(url):
  scrape_status = app.scrape_url(
    url,
    params={'formats': ['markdown', 'html']}
  )
  return scrape_status

scrape_website("https://langfuse.com")
```
