# Exa Integration

이 가이드에서는 [Langfuse](https://langfuse.com)를 [Exa](https://exa.ai/)와 통합하여 AI 검색 작업을 추적하는 방법을 보여드립니다. Langfuse의 트레이싱 기능을 활용하면 Exa 검색 함수의 입력, 출력, 실행 시간과 같은 세부 정보를 자동으로 캡처할 수 있습니다.

> **Exa란 무엇인가요?** [Exa](https://exa.ai/)는 LLM과 AI 애플리케이션을 위해 구축된 AI 기반 검색 API입니다. 기존 검색 엔진과 달리 Exa는 의미론적 의미를 이해하고, RAG(검색 증강 생성), 리서치, 콘텐츠 발견과 같은 AI 사용 사례에 적합한 고품질의 관련성 높은 결과를 검색하도록 설계되었습니다.

> **Langfuse란 무엇인가요?** [Langfuse](https://langfuse.com)는 팀이 LLM 애플리케이션에서 API 호출을 추적하고, 성능을 모니터링하고, 문제를 디버깅할 수 있도록 돕는 오픈 소스 AI 엔지니어링 플랫폼입니다.

## 예제 1: Exa `search` 트레이싱하기

Exa 검색 작업을 모니터링하기 위해 [Langfuse `@observe()` 데코레이터](https://langfuse.com/docs/sdk/python/decorators)를 사용합니다. 이 예제에서 `@observe()` 데코레이터는 `search_with_exa()` 함수의 입력, 출력, 실행 시간을 캡처합니다. Langfuse로 전송하는 데이터를 더 세밀하게 제어하려면 Python SDK를 사용해 [컨텍스트 관리자를 사용하거나 수동으로 옵저베이션을 생성](https://langfuse.com/docs/observability/sdk/python/instrumentation#custom-instrumentation)할 수 있습니다.

```python
from langfuse import observe

@observe(as_type="retriever")
def search_with_exa(query: str, num_results: int = 5):
    """Search the web using Exa AI and return results."""
    results = exa.search(
        query,
        num_results=num_results
    )
    return results

# Example: Search for information about Langfuse
search_results = search_with_exa("What is Langfuse and how does it help with LLM observability?")

# Display the results
for result in search_results.results:
    print(f"Title: {result.title}")
    print(f"URL: {result.url}")
    print()
```

## 예제 2: OpenAI와 함께 사용하는 Exa 검색

OpenAI로 검색 결과를 요약하는 좀 더 복잡한 워크플로도 추적할 수 있습니다. 여기서는 [Langfuse `@observe()` 데코레이터](https://langfuse.com/docs/sdk/python/decorators)를 사용해 Exa 검색과 OpenAI 생성을 하나의 트레이스로 묶습니다.

```python
import os
from langfuse.openai import OpenAI

@observe()
def search_and_summarize(query: str):

    # 1. Exa search with highlights
    @observe(as_type="retriever")
    def search_with_exa(query: str, num_results: int = 5):
        """Search the web using Exa AI and return results with highlights."""
        results = exa.search_and_contents(
            query,
            num_results=num_results,
            highlights=True
        )
        return results

    results = search_with_exa(query)

    # 2. Build context from search results
    context = "\n".join([
        f"{r.title}: {' '.join(r.highlights)}" for r in results.results
    ])

    # 3. Summarize with OpenAI
    client = OpenAI()
    resp = client.chat.completions.create(
        model="gpt-5-mini",
        messages=[
            {"role": "system", "content": "Summarize the following search results clearly and concisely."},
            {"role": "user", "content": context}
        ]
    )

    print("Summary:\n", resp.choices[0].message.content)

search_and_summarize("What is Langfuse and how does it help with LLM observability?")
```
