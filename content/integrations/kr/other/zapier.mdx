# Integration: Zapier

> [Zapier](https://zapier.com/)는 여러 앱과 서비스를 서로 연결하는 워크플로 자동화 플랫폼입니다.

Langfuse를 Zapier와 통합하여 LLM 엔지니어링 워크플로를 자동화하세요. Zapier를 통해 Langfuse Public API를 사용해 프로젝트를 관리하고, 프롬프트를 가져오고, Langfuse 계정의 이벤트를 기반으로 작업을 트리거할 수 있습니다.

## 사전 요구 사항

- 활성화된 Langfuse 계정(무료 Hobby 플랜 또는 유료 플랜에서 사용 가능)
- Zapier 계정

## 통합 범위

Zapier를 통해 [Langfuse Public API](/docs/api-and-data-platform/features/public-api)를 활용하면 다음을 할 수 있습니다.

- 프로젝트 관리
- 프롬프트 가져오기
- 트레이스, 평가, 기타 Langfuse 기능과 상호작용하기 위한 사용자 지정 API 요청 만들기

사용 가능한 API 엔드포인트에 대한 자세한 내용은 [Public API 문서](/docs/api-and-data-platform/features/public-api)를 참고하세요.

## 시작하기

<Steps>

### Step 1: Langfuse를 Zapier에 연결하기

1. Zapier의 [Apps 페이지](https://zapier.com/app/connections)로 이동합니다
2. "+ Add connection"을 클릭합니다
3. 대화 상자에서 "Langfuse"를 검색하여 선택합니다
4. "Add connection"을 클릭합니다
5. 새 브라우저 탭 또는 창이 열립니다. Langfuse Public Key와 Secret Key를 입력하고 데이터 리전을 선택합니다
6. "Yes, Continue to Langfuse"를 클릭합니다

이제 Langfuse 계정이 Zapier에 성공적으로 연결되었습니다.

### Step 2: Zap 생성하기

연결이 완료되면 Langfuse 트리거와 액션을 사용해 Zap을 생성할 수 있습니다.

**사용 가능한 트리거:**

- **New Project**: Langfuse 계정에 새 프로젝트가 생성될 때 트리거됩니다

**사용 가능한 액션:**

- **API Request (Beta)**: Langfuse API에 원시 HTTP 요청을 보내어 레코드를 생성 또는 업데이트하거나 데이터를 가져오는 등의 작업을 할 수 있습니다

</Steps>

## 예제 사용 사례

- **프롬프트 관리 자동화**: 새 프롬프트 버전이 생성될 때 문서에 자동으로 동기화하거나 Slack을 통해 팀에 알립니다
- **프로젝트 설정**: 새 Langfuse 프로젝트가 생성될 때 온보딩 워크플로를 트리거합니다
- **사용자 지정 통합**: API Request 액션을 사용해 트레이스, 점수, 기타 Langfuse 데이터와 상호작용하는 사용자 지정 워크플로를 구축합니다
