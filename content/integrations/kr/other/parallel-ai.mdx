# Parallel Integration

이 가이드에서는 [Langfuse](https://langfuse.com)를 [Parallel](https://parallel.ai/)과 통합하여 AI 작업을 추적하는 방법을 보여드립니다. Langfuse의 트레이싱 기능을 활용하면 Parallel 작업의 입력, 출력, 실행 시간과 같은 세부 정보를 자동으로 캡처할 수 있습니다.

> **Parallel이란 무엇인가요?** [Parallel](https://parallel.ai/)은 AI 에이전트, 애플리케이션, 워크플로를 오픈 인터넷에 연결하는 웹 검색 및 웹 에이전트 API 모음을 개발하여, 간단한 검색부터 복잡한 지식 작업까지 프로그래밍 가능한 작업을 가능하게 합니다.

> **Langfuse란 무엇인가요?** [Langfuse](https://langfuse.com)는 팀이 LLM 애플리케이션에서 API 호출을 추적하고, 성능을 모니터링하고, 문제를 디버깅할 수 있도록 돕는 오픈 소스 AI 엔지니어링 플랫폼입니다.

## 예제 1: Parallel Task API 트레이싱하기

Task API 요청을 모니터링하기 위해 [Langfuse `@observe()` 데코레이터](https://langfuse.com/docs/sdk/python/decorators)를 사용합니다. 이 예제에서 `@observe()` 데코레이터는 `parallel_task()` 함수의 입력, 출력, 실행 시간을 캡처합니다. Langfuse로 전송하는 데이터를 더 세밀하게 제어하려면 Python SDK를 사용해 [컨텍스트 관리자를 사용하거나 수동으로 옵저베이션을 생성](https://langfuse.com/docs/observability/sdk/python/instrumentation#custom-instrumentation)할 수 있습니다.

```python
import os
from parallel import Parallel
from parallel.types import TaskSpecParam
from langfuse import observe

client = Parallel(api_key=os.environ["PARALLEL_API_KEY"])

@observe(as_type="retriever")
def parallel_task(input: str):
    task_run = client.task_run.create(
        input=input,
        task_spec=TaskSpecParam(
            output_schema="The founding date of the company in the format MM-YYYY"
            ),
        processor="base"
    )
    print(f"Run ID: {task_run.run_id}")

    run_result = client.task_run.result(task_run.run_id, api_timeout=3600)
    print(run_result.output)

    return run_result.output

parallel_task("Langfuse")
```

## 예제 2: Parallel Chat API 트레이싱하기

[Langfuse OpenAI 래퍼](https://langfuse.com/integrations/model-providers/openai-py)를 사용해 Parallel Chat API와의 상호작용을 트레이싱할 수 있습니다.

```python
from langfuse.openai import OpenAI

client = OpenAI(
    api_key=os.environ["PARALLEL_API_KEY"],  # Your Parallel API key
    base_url="https://api.parallel.ai"  # Parallel's API beta endpoint
)

response = client.chat.completions.create(
    model="speed", # Parallel model name
    name="Parallel Chat",
    messages=[
        {"role": "user", "content": "What does Parallel Web Systems do?"}
    ],
    response_format={
        "type": "json_schema",
        "json_schema": {
            "name": "reasoning_schema",
            "schema": {
                "type": "object",
                "properties": {
                    "reasoning": {
                        "type": "string",
                        "description": "Think step by step to arrive at the answer",
                    },
                    "answer": {
                        "type": "string",
                        "description": "The direct answer to the question",
                    },
                    "citations": {
                        "type": "array",
                        "items": {"type": "string"},
                        "description": "Sources cited to support the answer",
                    },
                },
            },
        },
    },
)

print(response.choices[0].message.content)
```

## 예제 3: Parallel Search API와 OpenAI

OpenAI로 검색 결과를 요약하는 좀 더 복잡한 워크플로도 추적할 수 있습니다. 여기서는 [Langfuse `@observe()` 데코레이터](https://langfuse.com/docs/sdk/python/decorators)를 사용해 Parallel 검색과 OpenAI 생성을 하나의 트레이스로 묶습니다.

```python
import os
from parallel import Parallel
from langfuse.openai import OpenAI
from langfuse import observe


@observe()
def search_and_summarize(objective, search_queries):
    # 1. Parallel Search API
    parallel_client = Parallel(api_key=os.environ["PARALLEL_API_KEY"])

    @observe(as_type="retriever")
    def search_with_parallel(objective, search_queries, num_results: int = 5):
        """Search the web using Parallel and return results."""
        search = parallel_client.beta.search(
            objective=objective,
            search_queries=search_queries,
            processor="base",
            max_results=num_results,
            max_chars_per_result=6000
        )
        return search.results

    results = search_with_parallel(objective, search_queries)
    results_text = "\n\n".join(str(r) for r in results) if results else "No results."

    # 2. Summarize with OpenAI
    openai_client = OpenAI()
    resp = openai_client.chat.completions.create(
        model="gpt-5-mini",
        messages=[
            {"role": "system", "content": "Summarize the following search results clearly and concisely."},
            {"role": "user", "content": results_text}
        ]
    )
    return resp.choices[0].message.content

# Example usage
search_and_summarize(
    objective="Explain what Langfuse is and highlight its main features for LLM application observability.",
    search_queries=[
        "Langfuse LLM observability",
        "Langfuse features and documentation",
        "Langfuse tracing evaluations dashboards"
    ],
)
```
