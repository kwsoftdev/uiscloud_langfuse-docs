# Langfuse 트레이싱을 활성화 또는 비활성화하는 방법

모든 Langfuse SDK와 통합은 비침습적으로 설계되어 있습니다. 애플리케이션에 Langfuse 트레이싱을 추가하면서도 특정 환경에서만 활성화할 수 있습니다.

## 기본 동작

Langfuse SDK 중 하나를 사용하고 API 키가 설정되어 있으면 Langfuse 트레이싱은 기본적으로 **활성화**됩니다.

## 트레이싱 비활성화

`LANGFUSE_TRACING_ENABLED` 환경 변수를 `false`로 설정하여 트레이싱을 수동으로 비활성화할 수 있습니다.

```bash
export LANGFUSE_TRACING_ENABLED=false
```

또는 애플리케이션 설정에서 다음과 같이 지정할 수 있습니다.

```bash
LANGFUSE_TRACING_ENABLED=false
```
