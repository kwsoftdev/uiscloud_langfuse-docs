# Langfuse에서 조직이 보이지 않습니다

조직의 멤버로 추가되었는데도 대시보드에서 조직이 보이지 않는다면, 확인해야 할 몇 가지 일반적인 원인과 해결 방법이 있습니다.

## 올바른 Langfuse Cloud 리전에 있나요?

Langfuse는 여러 데이터 리전에서 운영되며 모든 데이터는 완전히 격리되어 있습니다. 조직이 생성된 것과 동일한 리전에 로그인되어 있는지 확인하세요.

[Langfuse Cloud 데이터 리전](/security/data-regions)에 대해 자세히 알아보세요.

## 올바른 이메일 주소를 사용하고 있나요?

조직 초대는 특정 이메일 주소로 발송됩니다. 초대를 받은 정확한 이메일 주소로 로그인하고 있는지 확인하세요.

**일반적인 문제:**

- SSO 제공업체(Google, GitHub 등)가 예상과 다른 이메일 주소를 사용할 수 있습니다
- 여러 이메일 주소를 가지고 있고 초대가 다른 주소로 발송되었을 수 있습니다
- 초대 이메일을 확인하여 어떤 주소로 발송되었는지 확인하세요

**현재 이메일 주소를 확인하는 방법:**

1. Langfuse UI 왼쪽 하단에서 프로필을 클릭하세요
2. 표시된 이메일 주소를 확인하세요
3. 조직 초대를 받은 이메일 주소와 일치하는지 확인하세요

## 여전히 문제가 있나요?

클라우드 리전과 이메일 주소를 확인했는데도 조직이 보이지 않는다면, 다음 정보와 함께 지원팀에 문의해 주세요:

- 이메일 주소
- 액세스하려는 조직 이름
- 조직이 속해 있다고 생각되는 클라우드 리전
- 표시되는 오류 메시지

**지원팀 문의:** [support@langfuse.com](mailto:support@langfuse.com)
