# Langfuse API 키는 어디에 있나요?

<Video
  src="https://static.langfuse.com/docs-videos/create-api-keys.mp4"
  aspectRatio={16 / 9}
  title="Where are my Langfuse API keys?"
/>

Langfuse 내에서 **Project --> Settings**로 이동하면 언제든지 API 키를 조회하거나 새로 생성할 수 있습니다.

새 프로젝트를 생성할 때도 Langfuse API 키를 받게 됩니다.
