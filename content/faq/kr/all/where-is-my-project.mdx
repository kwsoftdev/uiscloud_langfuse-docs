# 프로젝트를 찾을 수 없나요?

## 올바른 Langfuse 테넌트에 접속하고 있나요? (EU / US / Japan / HIPAA?)

Langfuse Cloud는 여러 데이터 리전에서 운영됩니다: [유럽연합(EU)](https://cloud.langfuse.com), [미국(US)](https://us.cloud.langfuse.com), [일본(Japan)](https://jp.cloud.langfuse.com), 그리고 별도의 [HIPAA 준수 미국 리전](https://hipaa.cloud.langfuse.com)입니다.
프로젝트를 생성한 데이터 리전에 로그인되어 있는지 확인하세요. 로그인 시 데이터 리전을 선택할 수 있습니다. 또한 URL 표시줄에서도 확인할 수 있습니다 (`cloud.langfuse.com` = EU, `us.cloud.langfuse.com` = US, `jp.cloud.langfuse.com` = Japan, `hipaa.cloud.langfuse.com` = HIPAA).

자세한 내용은 [Langfuse Cloud 데이터 리전](/security/data-regions)에서 확인하세요.

## 올바른 이메일 주소를 사용하고 있나요?

프로젝트에 초대를 받은 경우, 초대가 발송된 정확한 이메일 주소로 로그인하고 있는지 확인하세요. SSO(예: GitHub 또는 Google)로 로그인하는 경우, 이메일 주소가 예상과 다를 수 있습니다.

Langfuse UI 왼쪽 하단에서 사용자 이름을 클릭하면, 저희가 보유하고 있는 회원님의 이메일 주소를 확인할 수 있습니다.

## 프로젝트 소유자인가요?

조직 내 다른 사람이 프로젝트를 생성한 경우, 해당 사용자에게 재초대를 요청하거나 찾고 있는 프로젝트의 링크를 보내달라고 요청하세요.

## 그래도 프로젝트를 찾을 수 없나요?

support@langfuse.com으로 문의해 주시고, 가능한 한 자세한 정보(예: 프로젝트 이름, ID, 연결된 이메일, 프로젝트 참여자 등)를 함께 알려주세요.
