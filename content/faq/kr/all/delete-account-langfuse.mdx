# Langfuse 계정을 삭제하는 방법은?

## Langfuse Cloud

계정 설정(Account Settings) 페이지에서 계정을 삭제할 수 있습니다. 사용자 메뉴에서 Account Settings로 이동하세요.

조직의 유일한 소유자인 경우, 먼저 다른 사용자에게 소유권을 이전하거나 조직을 삭제해야 합니다 ([문서](/docs/administration/data-deletion#deleting-an-organization)).

## 셀프 호스팅 Langfuse

셀프 호스팅 Langfuse 인스턴스를 사용하고 있다면, 프로젝트와 관련된 데이터를 데이터베이스에서 직접 삭제해야 합니다.
