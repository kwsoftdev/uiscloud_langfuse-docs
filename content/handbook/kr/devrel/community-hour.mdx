# Langfuse 커뮤니티 아워

## Langfuse 커뮤니티 아워란 무엇인가요?

Langfuse 커뮤니티 아워는 사용자 질문에 답변하고 아이디어, 경험, 모범 사례를 공유하는 정기 모임입니다.

다음과 같은 주제로 사용자를 돕는 재미있고 실습 중심의 세션으로 진행합니다.

- Langfuse에 관한 질문
- 평가 및 테스트 설정 개선
- LLM 애플리케이션 구축에 관한 자유로운 대화

## Langfuse 커뮤니티 아워는 어떻게 일정을 잡나요?

1. **Langfuse Lu.ma 캘린더에 새 이벤트 생성**: https://luma.com/langfuse
   - 지난 이벤트는 "manage event" -> "more" -> "Clone Event"를 클릭해 복제할 수 있습니다.
   - 이벤트 2시간 전에 참가자에게 알림을 보내는 "Blast"를 추가합니다.
   - Lu.ma 초대 기능을 사용해 지난 커뮤니티 아워 참가자 전원을 초대합니다.

2. 이벤트용 Google Meet 링크를 만들기 위해 **새 Google Calendar 초대를 생성**합니다.
   - 호스트 승인 없이 누구나 참여할 수 있도록 이벤트 권한을 변경합니다(meeting access type = open).
   - Langfuse 팀의 모든 참여자를 미팅 호스트로 추가합니다.

3. **소셜 미디어에서 커뮤니티에 알림**을 보내 커뮤니티 아워를 안내합니다.
   - 소셜 툴 어셈블리를 사용해 월요일에 알림 게시물을 올립니다.
     - [예시 게시물](https://www.linkedin.com/feed/update/urn:li:share:7396844001577881601/)
   - Discord `#general` 채널에 짧은 메시지를 게시합니다.
     - [예시 메시지](https://discord.com/channels/1111061815649124414/1111063198989307964/1440763518046441725)
   - [소셜 미디어 이미지](https://www.figma.com/design/5mz0YBp9oQH3wfikRXVLOW/Langfuse-Media?node-id=3463-411&t=BtMggQdKPIilwCnh-4)
